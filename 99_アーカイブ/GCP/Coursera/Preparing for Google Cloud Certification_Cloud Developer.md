2. ウェブフックとして同期的に呼び出す。 
3. イベントでトリガーされたとき、非同期的に呼び出す。 
4. 特定の時刻に非同期的に呼び出す。
<details><div>
    答え：2,3
外部システムは、システム内のイベントに対するレスポンスで、ウェブフックとして関数を同期的に呼び出すことができます。Cloud Functions はまた、GCP サービスのイベントに対するレスポンスで、関数を非同期的にトリガーすることもできます。 
</div></details>

### Q. 2.
Cloud Functions は次のどのユースケースに最適ですか。 
1. 軽量、イベント駆動型、サーバーレス、マイクロサービス 
2. ポイントツーポイント接続が必要なアプリケーション 
3. 大規模で複雑なコードベースのアプリケーション

<details><div>
