## 工業簿記の種類
* 完全工業簿記
    + 原価計算を行い、その数値を複式簿記に組み込むもの
* 商的工業簿記
    + 原価計算を行わず、期末棚卸の結果から原価を把握するもの
## 原価計算の意義と目的
    原価計算基準では、原価計算の主たる目的として以下を挙げている。
    この目的を達成するために、原価計算の期間は通常1ヶ月となっており、これを原価計算期間と言う。
    ただし、製造業であっても、財務諸表を作成する会計期間は、1年で変わりはない。
* 財務諸表に表示するために必要な真実の原価を集計すること
* 価格計算に必要な原価資料を提供すること
* 経営管理者の各階層に対して、原価管理に必要な原価資料を提供すること
* 予算の編成ならびに予算統制のために必要な原価資料を提供すること
* 経営の基本計画を設定するに当たり、これに必要な原価情報を提供すること
## 原価計算の種類と形態
    原価計算は、計算の対象、目的、方法などにより、次のように分類される。
* 製造原価計算と総原価計算
* 実際原価計算と予定原価計算（見積原価計算、標準原価計算）
* 全部原価計算と直接原価計算
* 個別原価計算と総合原価計算
* 費目別計算、部門別計算、製品別計算
## 原価計算基準
* 原価は、経済価値の消費である
* 原価は、経営において作り出された一定の給付に転嫁される価値であり、その給付にかかわらせて、は握されたものである
* 原価は、経営目的に関連したものである
* 原価は、正常的なものである
## 給付
    経営が作り出す財貨をいい、それは経営の最終給付のみでなく、中間的給付をも意味する
## 製造原価と総原価
    製造原価と総原価は、次のような関係にある。
    なお、販売費及び一般管理費は、営業費とも言う。
* 製造原価
    + 製造に必要な原価で、材料費・労務費・経費の合計として捉えられる
* 総原価
    + 製造原価に、販売費及び一般管理費を加えたもの
* 製造原価は、製造直接費と製造間接費に分かれる。
    + 製造直接費（又は直接費）
        - 直接材料費
        - 直接労務費
        - 直接経費
    + 製造間接費（又は間接費）
        - 間接材料費
        - 間接労務費
        - 間接経費
## 原価計算の手続き
    価計算は、費目別、部門別、製品別の順番に進めるのが原則です。
ただし、このうち部門別計算は、主に製造間接費を製造部門ごとに配賦する作業になりますので、規模の小さい企業や、製造間接費が少ない場合は省略することもあります。
基本の流れは、まず費目別計算で材料費・労務費・経費を計算し、それぞれを製造直接費、製造間接費に分けます。
そして部門別計算で製造部門ごとに配賦し、最後に製品別計算で「仕掛品」と「製品」に振り替えるという手順になります。
### 費目別・部門別・製品別
    費目別計算、部門別計算、製品別計算は、次のように分類される。
* 費目別計算
    + 材料費計算
    + 労務費計算
    + 経費計算
    + 製造間接費計算
* 部門別計算
* 製品別計算
    + 個別原価計算
    + 総合原価計算
    + 標準原価計算
### 原価単位
    製品別計算のためには、原価を集計する一定の製品単位を定める。
    これが原価単位で、個数、時間数、度量衡単位など、業種の特質に応じて定める。

## 標準原価計算
### 原価標準
    原価計算基準では、原価計算制度を次の2つに分類し、定義している。
* 実際原価計算制度
    + 製品の実際原価を計算し、これを財務会計の主要帳簿に組み入れ、製品原価の計算と財務会計とが、実際原価をもって有機的に結合する原価計算制度
* 準原価計算制度
    + 製品の標準原価を計算し、これを財務会計の主要帳簿に組み入れ、製品原価の計算と財務会計とが、標準原価をもって有機的に結合する原価計算制度
### 標準原価計算
### 実際原価との差異計算
    予定原価による計算をした場合でも、あとで実際原価を計算して、その原価差異を処理します。ただし、一口に差異といっても、種類と程度により、その処理の方法も異なります。従って、差異が出たからといって、必ずしも当年度の売上原価に賦課するわけではありません。
#### 予定原価
    原価計算基準では、原価を実際原価、標準原価に分類しているが、現代では科学的、統計的根拠などに基づく標準原価と、その他の見積原価を分けて、次のような分類もされている。
* 実際原価
* 予定原価
    + 見積原価
    + 標準原価

## 製品原価計算
    原価計算基準では、「原価は、財務諸表上収益との対応関係に基づいて、製品原価と期間原価とに区別される」としています。
    このうち、製品原価は一定単位の製品に集計された原価。
    期間原価は、一定期間における発生額を当期の収益に直接対応させて把握した原価です。
### 個別原価計算
    個別原価計算は、顧客の注文によって個別仕様の製品を製造する、受注生産の製品原価を計算するときに使われるものです。
    実際に個別原価計算を行うときは、製造原価を製造直接費と製造間接費に分けて捉え、製造直接費は製品ごとに直課し、製造間接費は一定の基準で製品に配賦します。
#### 製造指図書と原価計算表
    顧客から注文を受けると、担当者が製造指図書を作成し、製造部門に対して製品の生産指示を出します。
    このあと、製造指図書に基づいて原価計算の担当が原価計算表を作り、製造原価の明細を記録して計算する、という流れです。
    製造原価の明細は、直接材料費、直接労務費、直接経費の明細と、製造間接費の配賦明細という形で記録されます。
* 製造指図書
    + 製造指図書番号
    + 製造品目・規格
    + 生産数量
    + 受注先
    + 製造着手日・完成予定日・完成日
    + 納期・引渡場所
* 原価計算表
    + 製造指図書番号・発行日
    + 製造着手日・完成日・引渡予定日
    + 受注先
    + 品名
    + 生産数量
    + 直接材料費の明細
    + 直接労務費の明細
    + 直接経費の明細
    + 製造間接費の配賦明細

### 部門別個別原価計算
    製造間接費を部門別に分けたあと、一旦製造部門にまとめてから仕掛品に振り替えるものです。
    部門別個別原価計算は、次の手順で進める。
1. 製造直接費、製造間接費に分ける
2. 製造直接費を、仕掛品に振り替える
3. 製造間接費に、次の処理をする
    * 製造部門費、補助部門費に分ける
    * 補助部門費を各製造部門に配賦する
    * 各製造部門費を、仕掛品に振り替える
4. 仕掛品を製品に振り替える
### 統合原価計算
    総合原価計算には、次のような種類がある。
#### 単純総合原価計算
    同種製品を反復連続的に生産する生産形態に適用する。
    製造間接費をそのまま仕掛品に振り替えるもの。
#### 等級別総合原価計算
    同一工程で同種製品を連続生産するが、その製品を形状、大きさ、品位等の等級で区別する場合に適用する。
    原価計算基準では、等級別総合原価計算について、次の2通りの方法を規定している。
* 等価係数を完成品総合原価に適用して、製品原価、単位原価を計算する
* 等価係数を原価要素別又は原価要素群別に適用して、製品原価、単位原価を計算する
#### 組別総合原価計算
    異種製品を組別に連続生産する生産形態に適用する
#### 工程別総合原価計算
    製造工程が2以上の連続工程に分けられ、工程ごとにその工程製品の総合原価を計算する場合に適用する
    累加法による工程別総合原価計算は、次のように進める。
1. 第1工程完成品総合原価を求め、第2工程の当月前工程費にする
2. 第2工程では前工程費を、直接材料費のように扱う
3. 第1工程、第2工程とも単純総合原価計算を行う

### 仕掛品と半製品
    製造工程の途中にある未完成品には、次の2種類がある。
* 仕掛品
    + その状態では販売できないもの
* 半製品
    + その状態でも販売ができるもの

### 補助元帳
    工業簿記では、次のような統制勘定、補助元帳が使われる。
* 材料 - 材料元帳
* 仕掛品 - 原価元帳
* 製品 - 製品元帳

### 仕損と減損
#### 仕損
    仕損とは加工に失敗することで、製造過程で何らかの原因により不合格品が出ることを言う。
    この不合格品を仕損品、ここでかかった原価を仕損費と言う。
* 異常な仕損
    + 通常予想される正常な範囲を超えた異常な仕損については、営業外費用又は特別損失として処理する。
* 正常な仕損
    + 補修指図書を発行する場合の会計処理は、次の流れになる。
1. 補修分を「仕掛品」に集める
2. 「仕掛品」を「仕損費」に振り替える
3. 「仕損費」を元の製造指図書の「仕掛品」に振り替える
    + これにより、「仕損費」は、直接経費として、元の製造指図書の「仕掛品」に賦課されたことになる。
* 仕損品が販売可能な場合は、以下の処理になる。
    + 主に材料として価値があれば、その分を直接材料費から差し引く
    + 主に加工による価値であれば、その分を加工費から差し引く
#### 減損
    原材料の蒸発、粉散などで発生する減少分。
#### 度外視法
    正常減損、正常仕損を、無視（度外視）して、その分を良品に負担させる方法を、度外視法と言う。
    本問は先入先出法で計算しているが、平均法も同じで、工程の途中で発生した場合は、正常減損がないものとして計算する。
* 工程の途中で発生
    + 完成品と月末仕掛品に負担させる
* 工程の終点で発生
    + 完成品に負担させる
### 作業屑の処理
    原材料を加工するにあたって、その屑が発生することがあります。
    これが「作業屑」で、このうち再利用や販売可能など、価値があるものについては、状況に合わせて会計処理が必要です。
    製品を特定できる場合は、直接材料費または製造原価から差し引く形で控除。特定できない場合は、製造間接費または該当部門の部門費から差し引くことになります。
#### 作業屑の会計処理
    作業屑は、次のように処理する。
* 製品を特定できる
    + 製品の直接材料費又は製造原価から差し引く
* 製品を特定できない
    + 部門別計算を行っている場合、該当部門の部門費から差し引く
    + 部門別計算を行っていない場合、製造間接費から差し引く
### 追加材料の原価配分
    追加材料の原価は、月末仕掛品原価には入れず、当月製造費用には入れることで、結果として完成品に配分されたことになる。
    追加材料の原価配分については、投入時点に従って、以下のように進める。
* 投入時点の工程の加工進捗度が、月末仕掛品加工進捗度以下の場合
    + 追加材料の直接材料費は、完成品と月末仕掛品に配分する
* 投入時点の工程の加工進捗度が、月末仕掛品加工進捗度超の場合
    + 追加材料の直接材料費は、完成品のみに配分する
* 工程を通じて平均的に投入する場合
    + 追加材料の直接材料費は、完成品と月末仕掛品に配分する
### 副産物の処理
    副産物は主製品に比べて重要性が低いことから、原価計算は行わず、主製品の原価から控除するのが一般的。
    ただし、軽微な副産物については、営業外収益とすることもある。

## 直接原価計算
    原価計算基準には、「原価は、集計される原価の範囲によって、全部原価と部分原価とに区別される」という区分の仕方もあります。
    全部原価は、全ての製造原価、又は販売費及び一般管理費を加えたもの。部分原価はその一部です。
    直接原価は変動原価とも呼ばれ、部分原価の中で最も重要なものに当たり、変動直接費と変動間接費のみを集計したものになります。
### 部分原価
* 計算目的ごとに計算した各種の原価
* 最も重要なものが、変動直接費と変動間接費を集計した直接原価（変動原価）

## 費目
### 材料費
    材料費は、主要材料費や買入部品費などの直接材料費と、補助材料費、工場消耗品費などの間接材料費に分かれます。
    消耗工具器具備品費は、耐用年数が1年未満又は小額で固定資産にならないものを購入したときの費用で、間接材料費の一つに分類されます。
    会計処理では、「直接材料費」は「仕掛品」を通して「製品」に直課され、「間接材料費」は一旦「製造間接費」に振り替え、そこから「仕掛品」を通して「製品」に配賦されます。
    材料費には、次の費用が含まれる。
* 直接材料費
    + 主要材料費（原料費）
    + 買入部品費
* 間接材料費
    + 補助材料費
    + 工場消耗品費
    + 消耗工具器具備品費
#### 直課
    直接、ある項目に賦課することを、“直課”と言う（又は、そのまま“賦課”とも言う）。これに対して、一定の基準で複数の項目に配分することを、“配賦”と言う。
#### 証憑
* 材料購入請求書
* 注文書
* 送り状
* 材料受入報告書
* 出庫票
#### 帳簿
* 材料仕入帳
* 材料仕訳帳
* 材料元帳
#### 材料の購入価額
    原価計算基準では、材料の購入原価（取得原価）は、購入代価と材料副費を加算した金額としています。このうち、材料副費については、引取費用のほかに、購入事務、検収、保管などの費用も挙げています。ただし、引取費用以外は、その一部又は全部を加算の対象にしなくても構いません。
##### 材料副費
    原価計算基準では、材料副費として以下を挙げている。
    引取費用は外部材料副費、その他は内部材料副費とも呼ばれる。
* 引取費用
    + 買入手数料、引取運賃、荷役費、保険料、関税等の買入に要した費用
* その他
    + 購入事務、検収、整理、選別、手入、保管等の費用
##### 予定配賦
    原価計算基準では、材料副費の一部又は全部は予定配賦率で計算できるとしている。また、予定配賦率は、一定期間の材料副費の予定総額を、その期間における材料の予定購入代価又は予定購入数量の総額で除して算定する、としている。
#### 材料の消費量
    原価計算基準では、材料の実際の消費量は、原則として「継続記録法」で計算するとしています。これは、材料の受入、払出、残高が常時把握できる方法ですが、その分手間がかかります。そこで、継続記録法による計算が難しい場合又はその必要がない場合は、「棚卸計算法」を適用してもよいことになっています。
#### 材料の消費単価
    原価計算基準では、同種材料の購入原価が異なる場合の消費単価の計算方法として、5つの方法を挙げています。移動平均法、総平均法と、先入先出法、後入先出法、個別法がそれです。このうち、後入先出法は企業会計の基準からは除外されています。
* 移動平均法
    + 材料の受入の都度、全在庫の単価を計算し直す方法
* 総平均法
    + 1ヶ月など一定期間の受入数量と金額から、単価を計算する方法
* 先入先出法
    + 先に受け入れた材料を先に払い出したとみなして、単価を計算する方法
* 後入先出法
    + 後に受け入れた材料を先に払い出したとみなして、単価を計算する方法
* 個別法
    + 材料を個別に管理することによって、それぞれの単価を計算する方法
#### 材料予定消費単価
    材料費の消費単価の計算では、予定価格を使う方法があります。
    これは、払出の単価に予定価格を使っておき、月末に実際価格との差異を調整する方法です。
* 材料消費価格差異 ＝（実際消費単価－予定消費単価）×実際消費量
#### 棚卸減耗
    正常な棚卸減耗とは、保管中に生じる材料の品質劣化など、通常発生するもの。これに対して、正常でない棚卸減耗とは、盗難、火災、自然災害などで、通常発生しないものを言う。
#### 材料費の記帳
    材料を払い出す場合、直接材料費は「仕掛品」に直課する形で振り替えます。一方、間接材料費については、一旦「製造間接費」にまとめて、そこから配賦の形で「仕掛品」に振り替えます。
* 製造間接費
    + 「製造間接費」には、間接材料費からの振替のほか、間接労務費、間接経費からの振替もある。
    こうして、「製造間接費」としてまとめた上で、一定の配賦方法に従って、「仕掛品」に振り替えられることになる。
### 労務費
    労務費は、直接労務費と間接労務費に分かれます。
    直接労務費は、直接工に対して支払った直接賃金など、そのまま製品の原価に直課されるもの。
    間接労務費は、間接工の賃金や、職員の給料や賞与手当、退職給付引当金繰入額など、また直接工であっても手待時間分の手待賃金などは間接労務費になります。
    会計処理では、材料費と同様、「直接労務費」は「仕掛品」に振り替えてから「製品」に直課され、「間接労務費」は「製造間接費」にまとめた後、「仕掛品」を通して「製品」に配賦されます。
* 直接労務費
    + 直接賃金
* 間接労務費
    + 間接作業賃金
    + 間接工賃金
    + 手待賃金
    + 休業賃金
    + 給料
    + 従業員賞与手当
    + 賞与引当金繰入額
    + 退職給付引当金繰入額
    + 法定福利費
#### 証憑
    労務費計算に使われる証ひょうには、次のようなものがある。
* 作業時間報告書
* 手待時間票
* 出勤簿
* タイムカード
* 賃金支払帳・給与支給帳
#### 帳簿
    労務費計算に使われる、主な帳簿は次の通り。
* （消費）賃金仕訳帳
* 一般仕訳帳
* 総勘定元帳
#### 時間の区分
    直接工の勤務時間は、次のように区分される。
* 勤務時間
    + 出勤簿に記録された時間で、途中休憩の休憩時間と、就業時間からなる
* 就業時間
    + 直接工の責任でない待ち時間である手待時間と、実際に働く実働時間からなる
* 手待時間
    + 直接工の責任ではない、機械の故障などによる待ち時間で、賃金の支払対象のため間接労務費となる
* 実働時間
    + 直接工が直接生産に従事する直接作業時間と、清掃などの間接作業時間からなる
* 直接作業時間
    + 直接工が直接加工作業などを行う加工時間と、作業にとりかかるための段取時間からなる
* 間接作業時間
    + 清掃や会議、朝礼、棚卸など、付加価値を生まない時間のため、間接労務費として取り扱う
#### 間接工
    間接工の賃金については、製造に直接関係しないことから、賃率は使わず、当月の要支払額を基に計算する。
#### 消費賃金
    予定賃率を使って労務費計算をする場合は、実際にかかった時間を基にします。
* 予定賃率
    + 予定賃率を使うと、実際の作業時間を掛けることで、その時点までの予定消費賃金を計算できる。従って、月の途中で製品が完成しても、その時点で製造原価が計算できる。
#### 消費賃金と支払賃金
    消費賃金と支払賃金には次の違いがあり、これは直接工、間接工のいずれにも適用される。
* 消費賃金
    + 原価計算期間に合わせて計算したもの
* 支払賃金
    + 支払賃金計算期間に合わせて計算したもの

## 非原価項目
    原価計算基準には、以下4分類の非原価項目が挙げられている。
* 経営目的に関連しない価値の減少
    + 投資資産、休止設備等の減価償却費、管理費、租税等
    + 寄付金等で経営目的に関連しない支出
    + 支払利息、社債発行費償却、支払保険料等の財務費用 など
* 異常な状態を原因とする価値の減少
    + 異常な仕損、減損、棚卸減耗
    + 火災、震災、盗難等の偶発的事故による損失
    + 違約金、損害賠償金、偶発債務損失、訴訟費
    + 固定資産売却損、異常な貸倒損失 など
* 税法上とくに認められている損失算入項目
    + 価格変動準備金繰入額
    + 通常の償却範囲額を超える額 など
* その他の利益剰余金に課する項目
    + 法人税、所得税、都道府県民税、市町村民税
    + 配当金、役員賞与金
    + 任意積立金繰入額
    + 建設利息償却 など
### 部門費
    部門費計算には、次のメリットがある。
* 各原価部門の管理状況がわかるので、原価管理が厳密になる
* 機械作業なら機械運転時間、手作業なら直接作業時間など、配賦基準を変えることで、より正確な配賦ができる
#### 第1次集計と第2次集計
    部門費計算では、最終的に部門費を製造部門に集中させる。この手続きは次の2段階で行う。
* 第1次集計
    + 部門費を部門個別費、部門共通費に分け、製造部門・補助部門に対して、前者は直課、後者は配賦する
* 第2次集計
    + 各補助部門に集まった補助部門費を、何らかの基準で、製造部門に配賦する
#### 部門個別費と部門共通費
    部門個別費と部門共通費には以下のようなものがある。
* 部門個別費
    + 補助材料費、特定部門所属の技師や職長の給料、専従従業員の間接賃金など
* 部門共通費
    + 共同利用の建物の減価償却費、固定資産税、保険料、機械の保険料、福利厚生費、福利施設負担費など
#### 部門共通費の配賦基準
* 建物減価償却費：建物占有面積
* 建物保険料　　：建物占有面積
* 建物固定資産税：建物占有面積
* 福利施設負担費：従業員数
* 福利厚生費　　：従業員数
* 機械保険料　　：機械帳簿価額
#### 補助部門費の製造部門への配賦
    直接配賦法と相互配賦法は、部門費計算の配賦に使われます。
    直接配賦法は、補助部門のサービスは全て製造部門に対するものとみなして、全額を直接、製造部門に配賦する方法。
    これに対して相互配賦法は、補助部門のサービスは製造部門のほかに他補助部門への提供もあるとして、計算を進めます。
    実際の現場では、補助部門のサービスは他補助部門へも提供されますので、相互配賦法の方がより正確な原価計算になります。
## 予算
### 固定予算と変動予算
    固定予算は、一定の操業度で計算した金額を、期間中適用するもの。つまり固定費だけということです。これに対して、変動予算は、固定費プラス変動費として、予算額を出します。
    これも固定費は一定の操業度で計算、変動費は実際の操業度に合わせて計算します。一定の操業度は、基準となるものですから基準操業度、実際の操業度は実際操業度と言います。
### 基準操業度
    基準操業度をどの程度にするかは、会社が決めることになる。
    ただし、次のいずれかを基に決めるのが一般的。
    このほか、最大限の生産能力による理論上達成可能な操業度を、理論的生産能力と呼ぶ。
* 実際的生産能力
    + 最大限の生産能力から、不可避な作業停止分を引いたもの
* 平均操業度
    + 将来の数年間の平均需要量に応えるための操業度
* 期待実際操業度
    + 予算期間中に予想される平均需要量に応えるための操業度
### 予定配賦
    製造部門の原価管理では、補助部門費の配賦額も含めて、予算を立てることが望ましい。
    このため、まず補助部門費を配賦して製造部門費の予算を立て、その上で、各製造部門の予定配賦額を決める。

#### 直接材料費
#### 直接労務費
#### 製造間接費
##### 実際配賦と予定配賦
    製造間接費の配賦には、実際配賦と予定配賦がある。
    実際配賦は製造間接費の実際の額を基にできる反面、金額の集計ができるまで計算できないこと、変動幅が大きいことなどのデメリットがある。
    これに対して、予定配賦は実際操業度さえ掴めれば計算できること、変動幅を抑えられるなどのメリットがあるため、実務では主にこちらが使われる。
##### 配賦差額
    製造間接費を予定配賦した場合、実際にかかった費用である実際発生額と、予定配賦額の間で差額が生じます。これが配賦差額で、これは、次のように予算差異と操業度差異に分けることができます。実際発生額と予算の差額が予算差異。見積もった予算に比べて、どれだけ実際の費用が多かったか、少なかったかと見る指標です。一方、予算と予定配賦額の差額が操業度差異。実際操業度が基準操業度からどれだけ離れたかを見る指標です。
* 操業度差異と予算
    + 予算の変動部分は、固定予算ではゼロ、変動予算では操業度×変動費率。
    + 一方、予定配賦額の変動部分は、該当製品の操業度に比例。
    + 従って、その差額である操業度差異は、操業度により変動することになり、実際操業度が基準操業度からどれだけ離れたかを見ることができる。
